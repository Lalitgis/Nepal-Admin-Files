# GIS Shapefile Repository

This repository contains GIS vector data and related spatial analysis files.  
The dataset includes shapefiles for mapping, visualization, and geospatial analysis.

## Contents

- Shapefile components (`.shp`, `.shx`, `.dbf`, `.prj`)
- Spatial datasets
- GIS project files
- Metadata and documentation

## Software

The data can be opened using:

- QGIS
- ArcMap/Pro
- Python and R
- Other GIS softwares

## Coordinate Reference System (CRS)

Please check the `.prj` file for projection information and EPSG details.

## Data ownership

The data is provided by the survey department of Nepal, and the owner of this repo doesn't guarantee the authenticity and accuracy of the data.
